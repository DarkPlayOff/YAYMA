# Список изменений

Все значимые изменения в проекте фиксируются в этом файле

## [2.3.0] - 2026-07-18

### Добавлено
- Настройки приложения на Android
- Доделано офлайн прослушивание треков
- Appimage и pkgbuild для Linux

### Исправлено
- Отображение треков в плейлистах
- Нормальный фикс воспроизведения треков
- Неадекватная работа фокуса аудио на Android
- Неадекватная работа WebView в Linux

### Изменено
- Улучшен и оптимизирован Vibe эффект
- Собсвенная подпись для Android билда
- Улучшен вид всех меню входа на Android
- Редизайн меню эквалайзера и эффектов

## [2.2.0] - 2026-07-13

### Добавлено
- Проверка обновлений
- Затухание звука при смене фокуса(звонки)
- Возможность загрузить треки в кеш(оффлайн режим пока не работает :))

### Исправлено
- Воспроизведение треков
- Невозможность закрыть меню лирики на Android
- Улучшена работа при нестабильном подключении к интернету


## [2.1.0] - 2026-06-25

### Добавлено
- Запуск «Моей Волны» по окончании очереди треков
- Настройка для полного закрытия приложения минуя трей
- Контекстное меню информации о треке на Android
- Карусель с обложками на Android
- Прогресс-бар и жест смены треков в мини-плеере на Android
- Возможность смены устройства воспроизведения звука (спасибо @duckinzzz)
- Новый способ логина ya.ru/device(полезно для Linux, где проблемы с WebView)

### Изменено
- Обновлен визуальный вид контекстного меню
- Перенос хранения токенов в SQLite
- Добавлена задержка перед открытием параметров «Моей Волны» (спасибо @duckinzzz)
- Внутренние улучшения и рефакторинг

### Исправлено
- Возможные сбои при авторизации
- Принудительная выгрузка клиента в фоновом режиме на Android
- Залипание подсветки при наведении на карточку трека
- Задержки перед началом воспроизведения треков
