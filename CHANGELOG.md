# Список изменений

Все значимые изменения в этом проекте фиксируются в этом файле

## [1.2.0] - 2026-04-20

### Список изменений
- Добавлена интеграция с Discord RPC
- Добавлена возможность скопировать ссылку на трек
- Исправлено открытие страницы управления аккаунтом
- Исправлено открытие страницы автора со страницы лайков
- Исправлено прыгание главного плеера при скрытии мини плеера
- Общие улучшения производительности

## [1.1.0] - 2026-04-18

### Список изменений
- Снято ограничение на вход для аккаунтов без подписки "Плюс"
- Добавлено полноэкранное раскрытие обложки альбома
- Добавлена кнопка для очистки кэша в настройках приложения
- Изменен цвет интерфейса по умолчанию
- Обновлена система навигации приложения
- Добавлено встраивание метаданных (обложка, автор) в скачанные FLAC треки
- Улучшена логика работы и отображения списка "лайков"
- Добавлено потоковое кэширование обложек
- Исправлено отображение обложек для пользовательских (UGC) треков
- Реализован корректный сброс состояния плеера после выхода из аккаунта
- Исправлена отправка обратной связи "Моей Волны" (`trackStarted`)
- Исправлена утечка памяти в аудио-буфере и уменьшен его размер
- Улучшен код работы с миниатюрами и проведена очистка логики жизненного цикла
