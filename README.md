## Предупреждение
**YAYMA** проект, созданный в целях обучения. Он использует неофициальное, незадокументированное API, не предназначенное для публичного использования. Пользоваться на свой страх и риск. Проект написан с помощью ИИ.

## Особенности
- **Отсутствие любой аналитики и слежки со стороны клиента**
- **Легковесный и быстрый**
- **Не Electron :)**
- **Управление плеером в предпросмотре окна**
- **Загрузка треков**
